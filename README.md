# IoT Ağ Trafiği Saldırı Tespiti - Slide Deck

Bu repo, IoT ağ trafiğinde yapay zekâ tabanlı saldırı tespiti projesinin HTML slide deck sürümüdür.

Microsite sürümü bu repo'ya dahil edilmemiştir. Bu repo yalnızca sunum/slayt deneyimi için hazırlanmıştır.

## Çalıştırma

Statik HTML olarak açılabilir:

```bash
python3 -m http.server 8766
```

Ardından:

```text
http://127.0.0.1:8766/
```

## İçerik

- `index.html`: 16:9 masaüstü sunum odaklı HTML slide deck.
